Adaptive Analysis of Heteroscedastic Multivariate Regression

2024
Dong · Lin · Cai · Yang
Journal of Statistical Software
Abstract

Multivariate regression is a widely used technique for large-scale statistical applications. In this paper, we propose a novel method for the multivariate regression model, called the Multivariate Weighted Lasso (MWL).

We consider the heteroscedasticity in the noise matrix and introduce an adjustable weight to calibrate the fitting residuals of different responses. The proposed method shows an advantage in complex multivariate regressions with heteroscedasticity. To implement the procedure, an efficient algorithm is provided based on multivariate coordinate descent. Theoretical guarantees are provided for the proposed method.

Numerical simulations and financial applications are conducted using the proposed and other existing methods. The results indicate that the proposed method performs well in both variable selection and coefficient estimation.

Method

Motivation

Standard multivariate lasso methods assume homoscedastic noise — i.e., equal variance across all response variables. In practice, especially in financial data, different response variables often exhibit very different noise levels (heteroscedasticity). Ignoring this leads to suboptimal variable selection and biased coefficient estimates.

Multivariate Weighted Lasso

The MWL introduces a diagonal weight matrix that adapts to the noise variance of each response. The weight is estimated from the residuals of an initial fit and used to rescale the penalty applied to each response, allowing the model to concentrate its regularization where signal-to-noise is poor.

minimize ‖Y − XB‖²_W + λ‖B‖₁

where W is the adaptive weight matrix calibrated to the heteroscedastic noise structure.

Algorithm

Estimation is carried out via multivariate coordinate descent, cycling through each coefficient of B while holding others fixed. The algorithm converges efficiently and scales well with the number of response variables.

Results

Simulations

Numerical simulations were conducted across multiple heteroscedastic settings, comparing MWL against standard multivariate lasso and other regularization approaches. MWL consistently achieved lower estimation error and higher true-positive variable selection rates, particularly in settings with high variance heterogeneity across responses.

Financial Application

The method was applied to financial return data where the heteroscedasticity assumption is well-motivated. MWL outperformed existing approaches in both variable selection stability and out-of-sample prediction accuracy.
